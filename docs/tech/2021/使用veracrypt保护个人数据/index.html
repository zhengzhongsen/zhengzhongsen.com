1. VeraCrypt 简介

VeraCrypt 是一款创建和维护动态加密卷的开源加密软件。

动态加密卷可以简单理解为一个加密文件，使用时需要将加密卷挂载到电脑上。挂载成功后，对用户而言，就是一个普通的磁盘，可以在里面创建文件目录，修改文件内容之类的。加密卷中的整个文件系统都是加密的，比如文件名、文件夹、每个文件内容和文件元信息等数据都是加密的。在使用过程中，它不会保存任何解密后的明文数据到磁盘上，即使是在加密卷被正确解密挂载到本地系统上，解密的数据也只是保存在内存中。这就保证了，就算攻击者强行关机拿到了磁盘上的加密卷文件，攻击者在不知道密码的情况下，也无法破解获取到加密卷里面的文件数据。

用户只有输入正确的密码或者密钥文件，才可以正常读取（解密）存储在加密卷里面的数据。使用数据的整个过程都无需用户介入，在加载读取数据时，数据自动解密，保存在内存上。在保存数据前，数据自动加密存储在动态加密卷中。

2. 如何使用VeraCrypt

2.1 安装 VeraCrypt

  • 从官网下载对应系统版本的软件安装包进行安装即可。

2.2 创建加密卷

  • 打开 VeraCrypt 软件，点击 Create Volume 开始创建加密卷，一路next默认即可。

  • 选择你加密卷文件的路径和文件名，比如我这里就创建了 D:\test\test.data 加密卷文件，这里的路径需要记住，后面挂载加密卷的时候需要用到这个文件。

  • 加密算法按照默认推荐的选项即可，安全性已经比较高了。

  • 加密卷大小可以根据你实际要加密保护的个人数据大小来设置，这里只是测试用途，就简单设置了10MB大小。如果你可能会有10多GB的文件数据需要加密保护，则可以将大小改为20GB之类的。

  • 设置加密卷的使用密码，这个密码设置完后一定要记住。后面使用这个加密卷的时候，需要这里设置的密码。

  • 在Windows系统上，推荐选择 NTFS 文件系统。

  • 在VeraCrypt的窗口上移动鼠标，保证密钥生产的随机性，最后点击 Format选项进行格式化文件。

  • 最后加密卷文件生成成功，加密卷文件路径为 D:\test\test.data。

2.3 使用加密卷

  • 打开 VeraCrypt 软件，随便选择一个没有被使用的驱动器，比如我这里选择了第1个驱动器盘符A。

  • 点击Select File...找到之前生成的加密卷文件 test.data

  • 点击 mount 挂载加密卷到本地磁盘。

  • 点击 mount 后会弹出密码窗口，输入之前设置的加密卷密码，点击OK即可。

  • 这时就可以看到加密卷文件 test.data 已经被挂载到驱动器A上面了，在计算机那里也可以看到磁盘A。

  • 后面就可以像普通磁盘一样，使用加密卷，比如说在里面创建个文件hello.txt。

  • 使用加密卷完毕后，点击Dismount卸载掉加密卷即可。

3. 一些建议

  • 一定要记住密码！！！密码一旦忘记了，加密卷里面的数据就无法解密了。
  • 加密卷文件的大小可以根据实际使用情况设置。
  • 定期备份重要文件。
